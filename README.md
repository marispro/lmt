- Vite v3, SCSS, Vue.js v3, Axios, Bootstrap v5


## Installation
1. Download project
2. rename ".env.example" to ".env" and set your database credentials
3. Run ```composer install```
4. Run ```npm install```
5. Compile assets by command ```vite dev```
6. Run migrations ```php artisan migrate```
