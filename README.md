## Tools used

- Vite v3, SCSS, Vue.js v3, Axios, Bootstrap v5


## Installation
1. Download project
2. Run ```composer install```
3. Run ```npm install```
4. Compile assets by command ```vite dev```
5. Run migrations ```php artisan migrate```
